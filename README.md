# ✨ Glowise

Glowise is an AI-powered skincare analysis web application built using Flask and Google Gemini AI.

## Features

- AI-powered skin analysis
- Skin type detection
- Personalized skincare routine
- Product recommendations
- Home remedies
- AI chatbot
- Analysis history

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Google Gemini API
- SQLite

## Installation

Clone the repository:

```bash
git clone https://github.com/Pragnya-Muchalambe/Glowise.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
